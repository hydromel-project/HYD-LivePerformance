HYD Live Performance

Setup Guide

1

Install via ReaPack

✓ Done

Scripts and web interfaces have been installed.

2

Run Setup Script

✓ Done

Toolbar button has been added.

3

Configure Web Server

Add a web browser interface in REAPER:

  1. Open Preferences (Ctrl+P)
  2. Navigate to Control/OSC/Web
  3. Click Add
  4. Select Web browser interface
  5. Set port to 9020
  6. Leave other settings as default
  7. Click OK, then Apply
Preferences → Control/OSC/Web → Add → Web browser interface → Port: 9020
4

Set Up Your Project

Create the required tracks in your REAPER project:

  1. Create a track named LYRICS for teleprompter text
  2. Create a track named SONGS for audio with metadata
  3. Add empty items to LYRICS track, put lyrics in Item Notes
  4. Create regions for each song section
5

Start the Server

Click the HYD Live Performance Server button in your toolbar, or run it from Actions.

Web Interface URLs

Teleprompter http://localhost:9020/Teleprompter.html
Now Playing http://localhost:9020/NowPlaying.html
Playlist http://localhost:9020/Playlist.html
Note: Links won't work until... You've completed Step 3 (web server config) and Step 5 (start server script).
