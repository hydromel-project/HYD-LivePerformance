HYD Live Performance

Setup Guide

Install via ReaPack

Scripts and web interfaces installed.

Run Setup Script

Toolbar button added.

Configure Web Server

Add a web interface in REAPER preferences:

  1. Open Preferences (Ctrl+P)
  2. Go to Control/OSC/Web
  3. Click Add
  4. Select Web browser interface
  5. Set port to 9020
  6. Click OK, then Apply

Project Setup

Create tracks in your REAPER project:

  1. Track named LYRICS — empty items with text in Item Notes
  2. Track named SONGS — audio files with metadata
  3. Create regions for song sections

Start Server

Click HYD Live Performance Server in your toolbar.

Web Interfaces

Teleprompter localhost:9020/Teleprompter.html
Now Playing localhost:9020/NowPlaying.html
Playlist localhost:9020/Playlist.html
Links work after completing step 3 (web server) and step 5 (start server).
